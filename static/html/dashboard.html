DyNAMiC Workbench (The Dynamic Nucleic Acid Mechanism Compiler) is an integrated development environment for developing dynamic DNA nanosystems in-silico. Workbench lets you begin with a behavioral description of a computational or assembly process, and then helps you automatically implement that process with DNA. It includes behavioral designers, sequence designers, and powerful sequence editing tools, as well as integration with industry-standard analysis tools.
