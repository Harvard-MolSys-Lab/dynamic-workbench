Getting Started

  • To create a new file:
    • Click New in the toolbar above
    • Enter a file name
    • Select a file type from the Create menu
  • To open an existing file:
    • Clicking on its name in the file tree to the left
DyNAMiC Workbench (The Dynamic Nucleic Acid Mechanism Compiler) is a project of Casey Grun and Justin Werfel in the Yin Group at Harvard University. DyNAMiC incorporates SpuriousDesign and the Pepper DNA Circuit compiler from the Erik Winfree group, NUPACK from the Niles Pierce Group,
